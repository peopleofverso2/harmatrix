# Harmatrix

Matrice tactile harmonique — prototype iPhone. Un instrument web autonome, en un seul fichier HTML, sans dépendance ni build.

## Lancer
Ouvre `harmatrix.html` dans un navigateur (idéalement Safari sur iPhone, en mode portrait). Touche le badge 🔊 pour activer le son.

## Fonctions
- **Matrice 8×8** : pads colorés (gamme / accord / tension / hors-gamme), tonique mise en avant.
- **Gestes** : vélocité selon la hauteur du toucher, glissé latéral = glide, glissé vers le bas = vibrato, multi-touch = accords.
- **Voix WebAudio** : Pad, Keys, Pluck, Bell, Cordes, Cuivres, boîte à rythmes, + échantillon micro (« Voice »).
- **FM Designer** : 6 algorithmes, 4 opérateurs (enveloppes AHR, feedback), sauvegarde locale.
- **Séquenceur** : 16 pas, 4 patterns (A–D), mode chanson, enregistrement live, générateur aléatoire musical.
- **Looper** + **enregistrement master** en .wav (capture tout ce que tu entends, partage iOS).
- **MIDI in/out** (Web MIDI), sauvegarde/chargement du projet en .json, retour haptique.

## Compatibilité
iPhone (Safari iOS), Chrome, Edge. Le MIDI nécessite iOS 16.4+ / Chrome / Edge.
